.. _details-struct_compound:

Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

Table of this type describes a Doxygen *compound* such as *class*, *namespace*, *group* etc.

Fields
------

.. index:: pair: variable; compoundKind
.. _doxid-field_compoundkind:

.. ref-code-block:: lua
	:class: doxyrest-title-code-block

	compoundKind

Holds a string describing compound type. Must be one of `CompoundKind`.

.. index:: pair: variable; id
.. _doxid-field_id:

.. ref-code-block:: lua
	:class: doxyrest-title-code-block

	id

Holds Doxygen identifier string, e.g. ``group__type_1gada12e3c9caedb0e18d384bfdc7e7def3``. This string can be used as a unique identifier for creating labels and reference links.

.. index:: pair: variable; name
.. _doxid-field_name:

.. ref-code-block:: lua
	:class: doxyrest-title-code-block

	name

Holds a string with the name of the compound.

.. index:: pair: variable; title
.. _doxid-field_title:

.. ref-code-block:: lua
	:class: doxyrest-title-code-block

	title

Holds a string with the title (if specified) or an empty string. Usually compound titles are not used unless this is a ``group`` compound.

.. index:: pair: variable; path
.. _doxid-field_path:

.. ref-code-block:: lua
	:class: doxyrest-title-code-block

	path

Holds a string with a forward-slash-separated path to the compound starting from the global namespace. For example, C++ class ``sys::win::WaitableTimer`` will have its path encoded as ``sys/win/WaitableTimer``. This field can be used to reconstruct a fully qualified name of the compound.

.. index:: pair: variable; briefDescription
.. _doxid-field_briefdescription:

.. ref-code-block:: lua
	:class: doxyrest-title-code-block

	briefDescription

Holds a `Description` table with the brief description of the compound.

.. index:: pair: variable; detailedDescription
.. _doxid-field_detaileddescription:

.. ref-code-block:: lua
	:class: doxyrest-title-code-block

	detailedDescription

Holds a `Description` table with the detailed description of the compound.

.. index:: pair: variable; location
.. _doxid-field_location:

.. ref-code-block:: lua
	:class: doxyrest-title-code-block

	location

Holds a `Location` table describing the place of declaration.
