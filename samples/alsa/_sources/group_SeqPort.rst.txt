	int
	snd_seq_create_port(

		:ref:`snd_seq_port_info_t<doxid-group___seq_port_1ga122b704fdab734223ef90b5e26f358e9>`* info
	)

create a sequencer port on the current client

Creates a sequencer port on the current client. The attributes of created port is specified in *info* argument.

The client field in *info* argument is overwritten with the current client id. The port id to be created can be specified via :ref:`snd_seq_port_info_set_port_specified <doxid-group___seq_port_1gafd4325db9167baf3e994a942577c2d5c>`. You can get the created port id by reading the port pointer via :ref:`snd_seq_port_info_get_port <doxid-group___seq_port_1ga90475c2db0acf804d8959058782493ed>`.

Each port has the capability bit-masks to specify the access capability of the port from other clients. The capability bit flags are defined as follows:

* :ref:`SND_SEQ_PORT_CAP_READ <doxid-group___seq_port_1ga81a7a5385af746364b757eb7911ec2f0>` Readable from this port

* :ref:`SND_SEQ_PORT_CAP_WRITE <doxid-group___seq_port_1gae8afb5ac8f546461b4bab25454972aeb>` Writable to this port.

* :ref:`SND_SEQ_PORT_CAP_SYNC_READ <doxid-group___seq_port_1gaebcc3dd1ad3b42d2305290feab04005a>` For synchronization (not implemented)

* :ref:`SND_SEQ_PORT_CAP_SYNC_WRITE <doxid-group___seq_port_1gabd27d66e2de0ff9556cc81cbec9ba7de>` For synchronization (not implemented)

* :ref:`SND_SEQ_PORT_CAP_DUPLEX <doxid-group___seq_port_1ga97da74dc7f0f295f000fadd1c50c1130>` Read/write duplex access is supported

* :ref:`SND_SEQ_PORT_CAP_SUBS_READ <doxid-group___seq_port_1ga4b8aa6cbeb3c99c3ad033f4ce69c06e2>` Read subscription is allowed

* :ref:`SND_SEQ_PORT_CAP_SUBS_WRITE <doxid-group___seq_port_1ga52ff1c51578166e67936cfb826c966c0>` Write subscription is allowed

* :ref:`SND_SEQ_PORT_CAP_NO_EXPORT <doxid-group___seq_port_1gaa0a63e34744ed9faedf64fe0b364bfd4>` Subscription management from 3rd client is disallowed

Each port has also the type bitmasks defined as follows:

* :ref:`SND_SEQ_PORT_TYPE_SPECIFIC <doxid-group___seq_port_1gaeb05a94bbdf256b857fcad28aeb2bb57>` Hardware specific port

* :ref:`SND_SEQ_PORT_TYPE_MIDI_GENERIC <doxid-group___seq_port_1ga0f89faac8be715b9e2d9d885c7f644a0>` Generic MIDI device

* :ref:`SND_SEQ_PORT_TYPE_MIDI_GM <doxid-group___seq_port_1ga7202cc7b9acc0d78f59c502d7be4d2d7>` General MIDI compatible device

* :ref:`SND_SEQ_PORT_TYPE_MIDI_GM2 <doxid-group___seq_port_1gabba1c9b09748d446bb6b20b0ae717984>` General MIDI 2 compatible device

* :ref:`SND_SEQ_PORT_TYPE_MIDI_GS <doxid-group___seq_port_1gadf72cb865e33a0ce1462f68303165537>` GS compatible device

* :ref:`SND_SEQ_PORT_TYPE_MIDI_XG <doxid-group___seq_port_1ga8b51348b38a543151dda96c73fbfb5d4>` XG compatible device

* :ref:`SND_SEQ_PORT_TYPE_MIDI_MT32 <doxid-group___seq_port_1ga12743fc5f31ab3718d5f86d88eef05fe>` MT-32 compatible device

* :ref:`SND_SEQ_PORT_TYPE_HARDWARE <doxid-group___seq_port_1ga7c7c4e97a41a4056c759f5398643a37f>` Implemented in hardware

* :ref:`SND_SEQ_PORT_TYPE_SOFTWARE <doxid-group___seq_port_1ga6b1a73e34cedbf61f039db614a50b27c>` Implemented in software

* :ref:`SND_SEQ_PORT_TYPE_SYNTHESIZER <doxid-group___seq_port_1gab873e7e613eee3af725f4a286c9c1cdf>` Generates sound

* :ref:`SND_SEQ_PORT_TYPE_PORT <doxid-group___seq_port_1ga3c9413dbab5e6c6a18235e8b05c8fb3c>` Connects to other device(s)

* :ref:`SND_SEQ_PORT_TYPE_APPLICATION <doxid-group___seq_port_1ga703a0651b941c583d5f8928f8f3b1b67>` Application (sequencer/editor)

A port may contain specific midi channels, midi voices and synth voices. These values could be zero as default.

query the next matching port

Queries the next matching port on the client specified in *info* argument. The search begins at the next port specified in port field of *info* argument. For finding the first port at a certain client, give -1.

If a matching port is found, its attributes are stored on *info* and function returns zero. Otherwise, a negative error code is returned.
