Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

Table of this type describes a block which `Description` table consists of.

Fields
------

.. index:: pair: variable; blockKind
.. _doxid-field_blockkind:

.. ref-code-block:: lua
	:class: doxyrest-title-code-block

	blockKind

Holds a string describing the type of the block. Must be one of `DocBlockKind`.

.. index:: pair: variable; title
.. _doxid-field_title_2:

.. ref-code-block:: lua
	:class: doxyrest-title-code-block

	title

Holds a string with the title of the block (empty string for untitled blocks).

.. index:: pair: variable; text
.. _doxid-field_text:

.. ref-code-block:: lua
	:class: doxyrest-title-code-block

	text

Holds a string with the plain-text contents of the block.

.. index:: pair: variable; childBlockList
.. _doxid-field_childblocklist:

.. ref-code-block:: lua
	:class: doxyrest-title-code-block

	childBlockList

Holds an array table with all the child blocks of the current block. Type of each element of the array is `DocBlock`.
