apr_status_t
apr_env_get(
    char** value,
    const char* envvar,
    apr_pool_t* pool
)

Get the value of an environment variable

Parameters:

value

the returned value, allocated from pool

envvar

the name of the environment variable

pool

where to allocate value and any temporary storage from
