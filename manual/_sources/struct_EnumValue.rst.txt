Detailed Documentation

Fields
------

.. index:: pair: variable; protectionKind
.. _doxid-field_protectionkind_2:

.. ref-code-block:: lua
	:class: doxyrest-title-code-block

	protectionKind

Holds a string describing protection of the enum value. Must be one of `ProtectionKind`. Usually it's just ``public``.

.. index:: pair: variable; id
.. _doxid-field_id_8:

.. ref-code-block:: lua
	:class: doxyrest-title-code-block

	id

Holds Doxygen identifier string, e.g. ``group__operators_1ggae611bfb523eb332069f2b7030accb66``. This string can be used as a unique identifier for creating labels and reference links.

.. index:: pair: variable; name
.. _doxid-field_name_4:

.. ref-code-block:: lua
	:class: doxyrest-title-code-block

	name

Holds a string with the name of the enum value.

.. index:: pair: variable; memberKind
.. _doxid-field_memberkind_2:

.. ref-code-block:: lua
	:class: doxyrest-title-code-block

	memberKind

Holds a string containing member kind; for ``EnumValue always set to ``"enummember"``

.. index:: pair: variable; initializer
.. _doxid-field_initializer:

.. ref-code-block:: lua
	:class: doxyrest-title-code-block

	initializer

Holds a `LinkedText` table with the manually defined value.

.. rubric:: Sample:

.. code-block:: cpp

	enum MyEnum
	{
		MyEnum_Value1,
		MyEnum_Value2 = 100,
	}

``initializer`` for ``MyEnum_Value1`` will be empty
(`LinkedText.isEmpty` will be set to ``true``).

``initializer`` for ``MyEnum_Value2`` will contain ``= 100``.
