Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

basic memory node structure

.. note::

	The next, ref and first_avail fields are available for use by the caller of :ref:`apr_allocator_alloc() <doxid-group__apr__allocator_1ga7452b0837c2148bd63109f89bbc0c3db>`, the remaining fields are read-only. The next field has to be used with caution and sensibly set when the memnode is passed back to :ref:`apr_allocator_free() <doxid-group__apr__allocator_1ga124e48339838213fd5e160a4125cb7e8>`. See :ref:`apr_allocator_free() <doxid-group__apr__allocator_1ga124e48339838213fd5e160a4125cb7e8>` for details. The ref and first_avail fields will be properly restored by :ref:`apr_allocator_free() <doxid-group__apr__allocator_1ga124e48339838213fd5e160a4125cb7e8>`.
