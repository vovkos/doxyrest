typedef int (*snd_hctl_compare_t)(
    const snd_hctl_elem_t *e1,
    const snd_hctl_elem_t *e2

Compare function for sorting HCTL elements.

Parameters:

e1

First element

e2

Second element

Returns:

-1 if e1 < e2, 0 if e1 == e2, 1 if e1 > e2
