Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

Table of this type describes a text (possibly) with injected hyperlinks.

Fields
------

.. index:: pair: variable; isEmpty
.. _doxid-field_isempty:

.. ref-code-block:: lua
	:class: doxyrest-title-code-block

	isEmpty

Holds ``true`` if the text is empty or ``false`` otherwise.

.. index:: pair: variable; plainText
.. _doxid-field_plaintext:

.. ref-code-block:: lua
	:class: doxyrest-title-code-block

	plainText

Holds a string with plain-text representation of the text, i.e. as if there were no any hyperlinks.

.. index:: pair: variable; refTextArray
.. _doxid-field_reftextarray:

.. ref-code-block:: lua
	:class: doxyrest-title-code-block

	refTextArray

Holds an array table of plain-text and hyperlink blocks constituting the full text. Type of each element of the array is `RefText`.
