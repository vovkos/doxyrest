.. ref-code-block:: c

	uint8_t bmRequestType

Request type.

Bits 0:4 determine recipient, see :ref:`libusb_request_recipient <doxid-group__libusb__misc_1ga20a4e8a729aa543fa742a5086596cbae>`. Bits 5:6 determine type, see :ref:`libusb_request_type <doxid-group__libusb__misc_1ga0b0933ae70744726cde11254c39fac91>`. Bit 7 determines data transfer direction, see :ref:`libusb_endpoint_direction <doxid-group__libusb__desc_1ga86c880af878493aa8f805c2aba654b8b>`.

	uint8_t bRequest

Request.

If the type bits of bmRequestType are equal to :ref:`LIBUSB_REQUEST_TYPE_STANDARD <doxid-group__libusb__misc_1gga0b0933ae70744726cde11254c39fac91a20eca62c34d2d25be7e1776510184209>` then this field refers to :ref:`libusb_standard_request <doxid-group__libusb__misc_1ga5b5912057c2d7e23b3ed5aa0c20236df>`. For other cases, use of this field is application-specific.
