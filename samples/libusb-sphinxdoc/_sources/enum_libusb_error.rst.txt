Detailed Documentation
~~~~~~~~~~~~~~~~~~~~~~

Error codes.

Most libusb functions return 0 on success or one of these codes on failure. You can call :ref:`libusb_error_name() <doxid-group__libusb__misc_1gacf319b823a1a1a25cc934d80db2f28c7>` to retrieve a string representation of an error code or :ref:`libusb_strerror() <doxid-group__libusb__misc_1gab977153d03d867a8b228a5921edaf457>` to get an end-user suitable description of an error code.
