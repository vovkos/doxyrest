enum apr_exit_why_e

Overview

#include <apr_thread_proc.h>

enum apr_exit_why_e {
    APR_PROC_EXIT        = 1,
    APR_PROC_SIGNAL      = 2,
    APR_PROC_SIGNAL_CORE = 4,
};

Detailed Documentation

Enum Values

APR_PROC_EXIT

process exited normally

APR_PROC_SIGNAL

process exited due to a signal

APR_PROC_SIGNAL_CORE

process exited and dumped a core file
