	int
	snd_pcm_extplug_create(
		:ref:`snd_pcm_extplug_t<doxid-group___p_c_m___ext_plug_1gaeddcbb8005ccafb008a75283d5d9e551>`* ext,
		const char* name,
		:ref:`snd_config_t<doxid-group___config_1ga1c20905af775ae77d04d1a5696f67985>`* root,
		:ref:`snd_config_t<doxid-group___config_1ga1c20905af775ae77d04d1a5696f67985>`* slave_conf,
		:ref:`snd_pcm_stream_t<doxid-group___p_c_m_1gac23b43ff55add78638e503b9cc892c24>` stream,
		int mode
	)

Create an extplug instance.

Creates the extplug instance based on the given handle. The slave_conf argument is mandatory, and usually taken from the config tree of the PCM plugin as "slave" config value. name, root, stream and mode arguments are the values used for opening the PCM.

The callback is the mandatory field of extplug handle. At least, start, stop and pointer callbacks must be set before calling this function.



.. rubric:: Parameters:

.. list-table::
	:widths: 20 80

	*
		- extplug

		- the extplug handle

	*
		- name

		- name of the PCM

	*
		- root

		- configuration tree root

	*
		- slave_conf

		- slave configuration root

	*
		- stream

		- stream direction

	*
		- mode

		- PCM open mode



.. rubric:: Returns:

0 if successful, or a negative error code
